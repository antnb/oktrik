Pernahkah anda mengalami saat sedang memasukan sebuah CD kedalam CD drive Komputer atau mungkin saat memasukan sebuah Flash disk Kedalam port USB sebuah Jendela Konfirmasi kemudian muncul dan Kemudian Menanyakan apa yang akan anda lakukan dengan files yang tersimpan dalam media tersebut( CD dan Flash Disk). Fitur tersebut dinamakan dengan Autoplay

Fitur Autoplay adalah fitur tambahan yang disediakan secara default oleh hampir semua operasi sitim windows baik windows xp, Windows Vista, Windows 7.

Meskipun cukup berguna, namun seiring waktu terkadang fitur Autoplay dan autorun sering kali cukup menjengkelkan. Karena seringkali jendela pop up Autoplay Maupun Autorun terkadang menggangu terutama apabila saat kita sedang bekerja dalam satu jendela windows.


 

Mematikan autorun

Secara default Autoplay Otomatis aktif dalam setting Group Policy window disetiap instalasi windows yang tersimpan dalam Registry

Selain terkadang Mengganggu, Autoplay pun sering kali menjadi salah satu penyebab masuknya virus kedalam komputer kita sehingga mengakibatkan terinfeksi sistim operasi oleh program jahat sehingga mengganggu kinerja komputer kita.

Autoplay dan autorun dikendalikan oleh sebuah file dengan extensi .inf dan seperti yang telah disebutkan diatas secara default sistim operasi pertama kali akan mencari File dengan extensi .inf tersebut dan apabila tidak ditemukan maka jendela pop up akan terbuka untuk melakukan konfirmasi kepada user tapi seting tersebut dapat kita matikan sehingga jendela autoplay tidak muncul

Namun seringkali disalahgunakan oleh orang orang yang tidak bertanggung jawab, sebagai contoh Berikut ini adalah sebuah file autorun.inf standar yang sering kita jumpai baik dalam sebuah CD Room yang berisi instalasi ataupun media Penyimpanan seperti Flash disk

[autorun]
open=setup.exe
icon=icon.ico
Seperti yang kita lihat, Maka media ( CD Rom maupun FlashDisk) tersebut dikonfigurasikan untuk Otomatis mengeksekusi sebuah file dengan nama setup.exe dan yang terjadi perintah OPEN tersebut dimodifikasi dan diarahkan kepada program jahat, seperti Keylogger yang akan merekam semua password yang diketik melalui tuts keyboard, Virus Yang merusak OS Dll

[autorun]
open=setup.exe
icon=icon.ico 

Konfigurasi Windows Autoplay tersimpan dalam Group Policy Editor dan secara default Windows Autoplay otomatis Aktif Atau tersetting Enabled. Dan tentu saja apabila kita akan mematikan fitur windows Autoplay harus kita disabled Terlebih dahulu Melalui Group Policy Editor. caranya pun terbilang cukup mudah

Tekan Logo Windows+R untuk membuka kolom RUN shortcut kemudian ketikan GPEDIT.MSC dan kemudian tekan enter. Maka Jendela Local Group Policy Editor Pun akan terbuka, pada bagian kolom sebelah kanan tabs WINDOWS CONFIGURATION double klik ADMINISTRATIVE TEMPLATE dan pilih system.

pada kolom sebelah kanan akan terbuka beberapa opsi, scrol pada bagian paling bawah maka akan ada menu "Turn off Autoplay" silahkan double klik menu tersebut dan silahkan pilih DISABLED

Sedangkan bagi user yang menggunakan Windows 7, 8 10 cara mendisable Windows Autoplay sedikit berbeda, Hal ini dikarenakan Path atau lokasi seting registry Windows Autoplay berbeda dengan windows XP, Pada windows 7 keatas Lokasi registry berada di WINDOWS COMPONENT  > AUTOPLAY POLICY

Serupa dengan cara mematikan Autoplay Pada Windows XP, Maka selanjutnya kita tinggal mematikan atau setting autoplay menjadi DISABLED
